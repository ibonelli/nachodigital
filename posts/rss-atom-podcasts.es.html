Hay varios servicios para recibir noticias (Facebook, Twitter, etc...) pero a mi el que mas me gusta es RSS/Atom debido a que son un standard que está abierto a todo el mundo. Además para leerlos un puede usar muchos programas, no una plataforma o servicio en particular.

Esto solo está relacionado lateralmente con Drupal. Debí aprender a distiguir entre ellos debido a un sitio que obtenía contenido consumiendo RSS y Atom feeds.

¿En que se diferencian? Básicamente hay dos formatos y uno para media:
  • El primero en aparecer fue RSS, tiene varias versiones y es el mas viejo.
  • Como algunos sentían que a RSS le faltaban cosas crearon Atom que es mas nuevo y supone ser una mejora sobre RSS.
  • Y para cerrar tenemos uno especialmente para media (audio, videos, etc...) llamado podcast, inicialmente impulsado por Apple.

Por que tantos? Ni idea... Pero supongo que tiene que ver con lo dificil que es ponernos de acuerdo. Pueden generar dolores de cabeza y cada uno tiene sus temitas, pero todos usan XML como lenguaje. Esto significa que para distiguirlos hay que ver el header de cada uno.

Finalmente quiero mencionar el servicio de Google feedburner. Este provee algunas interesantes opciones. Fundamentalmente una buena manera de "empujar"/enviar contenido a otras plataformas como Facebook o Twitter. Esto hace mucho mas sencillo la publicación de contenidos.

La relación de Drupal con esto es que pueden usar el módulo feeds para importar de una manera flexible el contenido publicado en forma de feeds en su sitio. Explicar como funciona eso lleva tiempo y es medio complicado, así que supera el alcance de este articulo.
